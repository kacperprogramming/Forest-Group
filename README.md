# Forest Group

This website was built as a summary project for Parts 1–4 of a web development course, covering the core frontend skills practiced throughout: semantic HTML5, CSS/SCSS architecture using the BEM methodology, responsive design with mobile-first media queries, Sass partials and variables, and vanilla JavaScript for DOM manipulation and form validation.
The project includes a multi-page layout (homepage, offer page, contact page) with features such as a sticky navigation bar with a mobile hamburger menu, a hero section with image optimization for different screen sizes, a pricing cards section, a contact form with client-side validation and a confirmation modal, an embedded Google Maps iframe, and a responsive footer with a sitemap.
Built without any frameworks or libraries — just HTML, SCSS, and plain JavaScript.
